# jobmap

A webapp for friends + classmates to add where they are working for coop

## Getting Started

```git clone git@github.com:rabeea-ahmad/jobmap.git```

```cd jobmap```

```npm install```

```mongod```

```node server.js```

```Go to localhost:3000```

```Profit```
